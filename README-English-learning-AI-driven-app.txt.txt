Project Overview

The AI-Driven English Learning Application is a group software development project aimed at helping users improve their English language skills through intelligent and adaptive learning features. The application focuses on usability, personalisation, and efficient system design.

Project Objectives

Develop a user-friendly English learning platform

Integrate AI-driven features to enhance learning and engagement

Provide a responsive interface for both mobile and web platforms

Ensure effective frontend, backend, and database integration

Key Features

AI-assisted English learning content

Interactive lessons and exercises

User progress tracking

Cross-platform support using a single shared codebase

System Architecture & Technology Stack

The application uses a single codebase to support both mobile and web platforms, improving maintainability and consistency.

Frontend: UI designed using Figma

Backend: Python, Java, and C#

Database: MySQL

Team Collaboration & Project Status

The project is developed collaboratively, with tasks divided between frontend and backend teams. Regular meetings are conducted to review progress and make technical decisions.

Current status:

UI designs completed and reviewed

Technology stack finalised

Development phase in progress

Conclusion

This project combines AI technology with educational design to deliver an effective English learning platform. The application demonstrates collaborative development, thoughtful system architecture, and a focus on user-centred design.
